# 모바일 와이어프레임 초안: 분양 전용 서비스

## 1. 문서 목적

- 화면 상세 기획을 실제 배치 수준으로 구체화한다.
- MVP 기준 핵심 화면의 정보 우선순위와 액션 위치를 정리한다.
- 디자이너와 개발자가 같은 구조를 기준으로 작업할 수 있게 한다.

## 2. 설계 원칙

- 모바일 우선으로 설계한다.
- 화면 첫 진입 시 `지금 행동 가능한 정보`가 가장 먼저 보여야 한다.
- 단지 소개보다 단계, 일정, 지원 가능성, 가격이 앞선다.
- 하단 탭 구조로 주요 탐색을 단순화한다.

## 3. 공통 UI 구조

### 하단 탭
- 홈
- 분양 찾기
- 일정
- 관심
- 마이

### 공통 카드 원칙
- 제목
- 상태 배지
- 핵심 숫자 1~2개
- 다음 일정
- 저장 또는 상세 액션

### 공통 배지
- 분양 예정
- 공고 공개
- 특별공급 진행
- 1순위 진행
- 2순위 진행
- 발표 완료
- 계약 진행
- 입주 예정

## 4. 온보딩

### 목적
- 첫 진입 시 서비스 가치를 전달하고, 맞춤 추천에 필요한 조건을 자연스럽게 수집한다.

### 1단계: 서비스 소개 와이어프레임

```text
+--------------------------------------------------+
|                                                    |
|            [일러스트/아이콘]                       |
|                                                    |
|     분양과 청약에 필요한 정보만 모았어요           |
|                                                    |
|     복잡한 부동산 정보 대신                        |
|     지금 청약 가능한 단지부터 보여드려요           |
|                                                    |
|                 ● ○ ○                              |
|                                                    |
|                                          [다음]    |
|     [건너뛰기]                                     |
+--------------------------------------------------+
```

### 2단계: 관심 지역 선택 와이어프레임

```text
+--------------------------------------------------+
|                                          [건너뛰기]|
|--------------------------------------------------|
| 어느 지역에 관심 있으세요?                         |
| (여러 개 선택 가능)                                |
|--------------------------------------------------|
| [서울]  [경기]  [인천]                             |
| [부산]  [대구]  [대전]                             |
| [광주]  [울산]  [세종]                             |
| [강원]  [충북]  [충남]                             |
| [전북]  [전남]  [경북]                             |
| [경남]  [제주]                                     |
|--------------------------------------------------|
| ████░░░░░░░░░░░░░░░  2/5                          |
|                                          [다음]    |
+--------------------------------------------------+
```

### 3단계: 예산/조건 입력 와이어프레임

```text
+--------------------------------------------------+
|                                          [건너뛰기]|
|--------------------------------------------------|
| 예산은 어느 정도인가요?                            |
| [최소] --------slider-------- [최대]               |
|                   6억                              |
|--------------------------------------------------|
| 무주택인가요?          [예]  [아니오]              |
| 세대주인가요?          [예]  [아니오]              |
| 청약통장이 있나요?     [예]  [아니오]              |
|--------------------------------------------------|
| 특별공급 대상인가요?                               |
| [신혼부부] [생애최초] [다자녀] [해당없음]          |
|--------------------------------------------------|
| ████████████░░░░░░░  4/5                          |
|                                        [완료]      |
+--------------------------------------------------+
```

### 완료 화면 와이어프레임

```text
+--------------------------------------------------+
|                                                    |
|            [체크 아이콘]                           |
|                                                    |
|        조건 설정이 완료되었어요!                   |
|                                                    |
|     서울 · 경기 / 6억 이하 / 신혼부부              |
|                                                    |
|                                                    |
|         [맞춤 추천 단지 보러가기]                  |
|                                                    |
+--------------------------------------------------+
```

### 핵심 포인트
- 전체 플로우 1분 이내 완료를 목표로 한다.
- 모든 단계에서 건너뛰기가 가능하다.
- 완료 후 바로 개인화된 홈 화면으로 연결해 첫 경험의 가치를 높인다.

## 4.1 홈

### 목적
- 지금 볼 만한 단지를 가장 빠르게 발견하게 한다.

### 와이어프레임

```text
+--------------------------------------------------+
| [로고]                    [지역]      [알림]     |
|--------------------------------------------------|
| [검색: 단지명, 지역명 검색]                       |
|--------------------------------------------------|
| [내 조건 요약 카드]                              |
| 거주지: 서울 / 무주택 / 신혼부부 / 예산 8억 이하 |
| [조건 수정]                                      |
|--------------------------------------------------|
| 지금 청약 가능한 단지                  [더보기]  |
| [특별공급 진행] 강동 리버파크         D-1       |
| 서울 강동구 / 민간분양 / 84A / 7.9억            |
| [관심] [상세]                                    |
|--------------------------------------------------|
| 곧 모집공고 나오는 단지               [더보기]  |
| [분양 예정] 마포 센트럴뷰            D-5        |
| 서울 마포구 / 공공분양 / 59 / 미정              |
|--------------------------------------------------|
| 내 조건에 맞는 추천 단지               [더보기] |
| 카드 1                                            |
| 카드 2                                            |
|--------------------------------------------------|
| 일정 임박 단지                         [더보기] |
| 카드 1                                            |
+--------------------------------------------------+
| 홈 | 분양 찾기 | 일정 | 관심 | 마이             |
+--------------------------------------------------+
```

### 핵심 포인트
- 홈 첫 화면에서 검색보다 `현재 접수 가능 단지`가 더 중요하다.
- 내 조건 카드가 있어야 추천과 자격 진단 흐름이 자연스럽다.
- 각 섹션은 2~3개만 미리 보여주고 전체는 리스트로 이동시킨다.

## 5. 분양 찾기

### 목적
- 단계와 조건 기반으로 단지를 빠르게 추리는 메인 탐색 화면

### 와이어프레임

```text
+--------------------------------------------------+
| [뒤로] 분양 찾기                     [필터]      |
|--------------------------------------------------|
| [검색: 지역, 단지명]                             |
|--------------------------------------------------|
| [분양 예정] [공고 공개] [특별공급] [1순위]       |
| [2순위] [발표] [계약]                            |
|--------------------------------------------------|
| 적용 필터: 서울, 특별공급, 8억 이하        [초기화]|
| 결과 128건                                       |
| 정렬: 접수 가능 순                               |
|--------------------------------------------------|
| [특별공급 진행] 성수 더하이츠          D-1       |
| 서울 성동구 / 민간분양 / 74,84 / 8.2억          |
| 공급유형: 신혼부부, 생애최초                     |
| [관심] [비교]                                    |
|--------------------------------------------------|
| [공고 공개] 하남 레이크뷰             D-3        |
| 경기 하남시 / 공공분양 / 59 / 5.4억             |
| 공급유형: 일반공급, 생애최초                     |
| [관심] [비교]                                    |
+--------------------------------------------------+
| 홈 | 분양 찾기 | 일정 | 관심 | 마이             |
+--------------------------------------------------+
```

### 핵심 포인트
- 분양 단계 칩은 상단 고정이 적합하다.
- 카드에서 상세 진입 전에도 단계, 가격, 일정, 공급 유형까지 파악 가능해야 한다.
- 사용자가 일반 부동산 서비스처럼 느끼지 않도록 방 수나 옵션 중심이 아니라 청약 실행 정보 중심으로 보여준다.

## 5.1 분양 찾기 (지도 뷰)

### 목적
- 관심 지역의 분양 단지를 지도 위에서 직관적으로 탐색한다.

### 와이어프레임

```text
+--------------------------------------------------+
| [뒤로] 분양 찾기              [리스트] [필터]     |
|--------------------------------------------------|
| [검색: 지역, 단지명]                              |
|--------------------------------------------------|
| [분양 예정] [공고 공개] [특별공급] [1순위]        |
|--------------------------------------------------|
|                                                    |
|         [지도 영역]                                |
|                                                    |
|    🟢마포                                          |
|         센트럴뷰                                   |
|                    🔴성수        🟡강동             |
|                    더하이츠      리버파크           |
|                                                    |
|         🔵3                                        |
|                                                    |
|  범례: 🟢예정 🟠공고 🔴특별 🟡접수 🔵클러스터    |
|  이 지역 12건                    [이 지역 재검색]  |
|--------------------------------------------------|
| ━━━━━━━━━━━━━━━━━━ (슬라이드 핸들) ━━━━━━━━━━━━━  |
| [특별공급 진행] 성수 더하이츠          D-1        |
| 서울 성동구 / 민간분양 / 8.2억                    |
| [관심] [상세]                                     |
|--------------------------------------------------|
| [공고 공개] 마포 센트럴뷰              D-3        |
| 서울 마포구 / 공공분양 / 5.4억                    |
+--------------------------------------------------+
| 홈 | 분양 찾기 | 일정 | 관심 | 마이              |
+--------------------------------------------------+
```

### 핵심 포인트
- 지도 마커 색상은 분양 단계 배지 색상과 동일하게 통일한다.
- 하단 슬라이드 패널을 위로 올리면 리스트 뷰처럼 확장된다.
- 지도 영역 이동 시 「이 지역 재검색」 버튼으로 결과를 갱신한다.
- 상단 필터 칩은 리스트 뷰와 동일하게 적용된다.

## 6. 필터 바텀시트

### 목적
- 분양 전용 조건을 빠르게 조합하게 한다.

### 와이어프레임

```text
+--------------------------------------------------+
| 필터                                      [닫기] |
|--------------------------------------------------|
| 분양 단계                                         |
| [ ] 분양 예정  [ ] 공고 공개  [x] 특별공급       |
| [ ] 1순위     [ ] 2순위     [ ] 발표             |
|--------------------------------------------------|
| 지역                                              |
| [서울] [경기] [인천] [부산]                       |
|--------------------------------------------------|
| 공급 유형                                         |
| [민간분양] [공공분양]                             |
|--------------------------------------------------|
| 공급 대상                                         |
| [일반공급] [신혼부부] [생애최초] [다자녀]        |
|--------------------------------------------------|
| 가격 범위                                         |
| [최소] ----slider---- [최대]                      |
|--------------------------------------------------|
| 면적                                              |
| [59] [74] [84] [기타]                             |
|--------------------------------------------------|
| 입주 예정 시기                                    |
| [2026] [2027] [2028+]                             |
|--------------------------------------------------|
| [초기화]                         [결과 128건 보기]|
+--------------------------------------------------+
```

### 핵심 포인트
- 필터는 정보량이 많아 바텀시트 내 스크롤 구조가 적합하다.
- 사용자는 대부분 단계, 지역, 가격, 공급 대상만 먼저 쓸 가능성이 높다.

## 7. 단지 상세

### 목적
- 청약 결정을 위한 핵심 정보를 가장 빠르게 전달한다.

### 와이어프레임

```text
+--------------------------------------------------+
| [뒤로] 단지 상세                     [공유]      |
|--------------------------------------------------|
| 강동 리버파크                                     |
| 서울 강동구                                       |
| [특별공급 진행] D-1                               |
| [관심 저장] [비교 추가]                           |
|--------------------------------------------------|
| 다음 일정                                         |
| 특별공급 접수: 2026.03.10                         |
| 1순위 접수:   2026.03.11                          |
| 당첨 발표:    2026.03.18                          |
|--------------------------------------------------|
| 내 조건 기준                                      |
| 신혼부부 특별공급 가능성이 있어요                 |
| 확인 필요: 청약통장 가입 기간                     |
| [자세히 진단하기]                                 |
|--------------------------------------------------|
| 분양가 요약                                       |
| 59A: 6.8억                                        |
| 74A: 7.4억                                        |
| 84A: 7.9억                                        |
| 계약금 10% / 중도금 60% / 잔금 30%                |
|--------------------------------------------------|
| 규제 정보                                         |
| [상한제] [전매제한 3년] [실거주 2년] [조정지역]   |
| [중도금 대출 가능]                                |
|--------------------------------------------------|
| 경쟁률 참고                                       |
| 이 지역 유사 단지 평균 경쟁률: 12.3:1             |
| 과거 당첨 가점 범위: 58~72점                      |
| 내 가점(65점) 기준: 가능성 있음                   |
| 가점제 40% / 추첨제 60%                           |
| [가점 계산하기]                                   |
|--------------------------------------------------|
| 공급 정보                                         |
| 총 820세대 / 일반 320 / 특별 500                 |
| 민간분양 / 신혼부부, 생애최초, 다자녀            |
|--------------------------------------------------|
| 자금 계획 요약                                    |
| 84A 기준 분양가 7.9억                             |
| 계약금 7,900만 / 중도금 4.74억 / 잔금 2.37억     |
| 계약: 2026.04 / 중도금: 2026.10~ / 잔금: 2028.06 |
| [자금 시뮬레이션 열기]                            |
|--------------------------------------------------|
| 일정 타임라인                                     |
| 공고 공개 -> 특별공급 -> 1순위 -> 발표 -> 계약    |
|--------------------------------------------------|
| 입지 요약                                         |
| 지하철 8분 / 초등학교 도보 5분 / 대형마트 10분    |
|--------------------------------------------------|
| 유의사항                                          |
| 거주지역 우선공급 여부 확인 필요                  |
| [공고문 원문 보기]                                |
+--------------------------------------------------+
| [알림 설정]                     [청약 준비 체크] |
+--------------------------------------------------+
```

### 핵심 포인트
- 상세 상단 1스크린 안에 `현재 단계`, `다음 일정`, `내 조건 기준 결과`, `분양가 요약`이 보여야 한다.
- 단지 홍보성 카피보다 실질 의사결정 정보가 우선이다.

## 8. 청약 자격 진단

### 목적
- 사용자가 빠르게 자신의 조건을 입력하고 결과를 받게 한다.

### 1단계 입력 와이어프레임

```text
+--------------------------------------------------+
| [뒤로] 청약 자격 진단                             |
|--------------------------------------------------|
| 1. 거주 지역은 어디인가요?                        |
| [서울] [경기] [인천] [기타]                       |
|--------------------------------------------------|
| 2. 현재 무주택인가요?                             |
| [예] [아니오]                                     |
|--------------------------------------------------|
| 3. 세대주인가요?                                  |
| [예] [아니오]                                     |
|--------------------------------------------------|
| 4. 청약통장이 있나요?                             |
| [예] [아니오]                                     |
|--------------------------------------------------|
| 5. 특별공급 대상인가요?                           |
| [신혼부부] [생애최초] [다자녀] [해당 없음]       |
|--------------------------------------------------|
| [다음]                                            |
+--------------------------------------------------+
```

### 결과 와이어프레임

```text
+--------------------------------------------------+
| 진단 결과                                         |
|--------------------------------------------------|
| 현재 기준으로 볼 만한 공급 유형                   |
| - 신혼부부 특별공급                               |
| - 생애최초 특별공급 가능성 검토 필요              |
|--------------------------------------------------|
| 추가 확인 필요                                    |
| - 청약통장 가입 기간                              |
| - 소득 요건                                       |
|--------------------------------------------------|
| 추천 단지                                         |
| 카드 1                                            |
| 카드 2                                            |
|--------------------------------------------------|
| [내 조건 저장]                 [추천 단지 보러가기]|
+--------------------------------------------------+
```

### 핵심 포인트
- 질문 수를 줄여 이탈을 막아야 한다.
- 결과는 확정 판정이 아니라 `가능성 + 추가 확인 포인트` 형태가 안전하다.

## 8.1 청약 가점 계산기

### 목적
- 자신의 청약 가점을 빠르게 계산하고, 당첨 가능성을 가늠한다.

### 입력 와이어프레임

```text
+--------------------------------------------------+
| [뒤로] 청약 가점 계산기                           |
|--------------------------------------------------|
```

## 8.2 비교함

### 목적

```text
| 지역      | 서울 강동   | 서울 성동   | 서울 마포 |
```

- 최대 5개까지 비교 가능하며, 좌우 스크롤로 열람한다.

## 12. MVP 제작 우선순위

### 1차 디자인 대상

- 홈

### 2차 디자인 대상
- 비교함

- 자금 시뮬레이션 상세

## 13. 디자이너 전달 메모

- 일반 부동산 앱 느낌을 줄이기 위해 `매물 카드`처럼 보이는 요소를 줄인다.
- 숫자와 배지, 일정 중심 계층 구조가 중요하다.
- 상태 변화가 많은 서비스라 시각적 상태 체계가 필요하다.
- 화면마다 `다음 행동` 버튼이 분명해야 한다.

## 14. 개발 전달 메모

- 홈 섹션은 동일 카드 컴포넌트 재사용 구조가 적합하다.
- 리스트 카드와 관심 카드의 정보 구조는 최대한 통일한다.
- 단계 배지와 일정 타임라인은 별도 공통 컴포넌트화가 유리하다.
- 단지 상세는 상단 요약 영역과 하단 탭형 정보 섹션으로 분리 가능하다.
- 규제 배지 바는 공통 컴포넌트로 단지 상세/비교함에서 재사용한다.
- 가점 점수 바/게이지는 가점 계산기와 단지 상세의 경쟁률 블록에서 공유한다.
- 지도 마커 색상 체계와 단계 배지 색상 체계를 통일한다.
- 비교함은 모바일에서 좌우 스크롤, 데스크톱에서는 테이블 형태를 지원한다.
