# 작업 계획: AGENTS 규칙 추가

## 목표
- 저장소 운영 규칙을 `AGENTS.md`에 문서화한다.
- 요청된 작업 계획/작업 로그 규칙을 명시한다.
- 기존 기획 문서 기반으로 컨텍스트 확인 원칙과 유지보수 원칙을 포함한다.

## 참고 문서
- `CLAUDE.md`
- `PRD.md`
- `TECH_ARCHITECTURE.md`
- `DATA_MODEL.md`
- `SCREEN_SPEC.md`
- `WIREFRAMES.md`

## 작업 범위
- 루트에 `AGENTS.md` 신규 작성
- `ai/plan`에 본 계획 문서 추가
- `ai/work`에 완료 로그 문서 추가

## 구현 단계
1. 기존 문서 구조와 프로젝트 방향을 확인한다.
2. 요청한 규칙을 실행 가능한 형태로 `AGENTS.md`에 정리한다.
3. 완료 후 변경 사항과 확인 내용을 `ai/work`에 기록한다.

## 유의 사항
- 현재 저장소는 기획 문서 중심이므로 문서 참조 규칙이 중요하다.
- 중복 로직 및 기능 상충 방지를 위한 컨텍스트 확인 절차를 명확히 적는다.
